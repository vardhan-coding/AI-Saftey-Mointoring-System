# AI Safety Monitoring System

## Overview
This project is an AI-based safety monitoring system developed using Python to simulate intelligent detection of hazardous environmental conditions. The system uses logical analysis to identify potential risks and generate alerts.

## Problem
In many industrial environments, unsafe conditions such as gas leaks, temperature changes, or abnormal situations may go unnoticed, leading to safety risks.

## Solution
This project implements an AI-based monitoring approach that analyzes environmental parameters and detects abnormal conditions to trigger safety alerts.

## Key Features
- AI-based hazard detection simulation
- Environmental condition monitoring
- Automated safety alerts
- Intelligent decision-making logic

## Technologies Used
- Python
- Basic AI Logic

## How to Run
1. Install Python.
2. Clone or download this repository.
3. Run the program using:
   python main.py
